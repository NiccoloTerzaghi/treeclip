Usage: treeclip [--exclude=folder1,folder2,...] [--help|-h]

Options:
  --exclude=folders   Comma-separated list of folder names to collapse (overrides .treeclipignore and defaults)
  --help, -h          Show this help message and exit

Ignore list:
  - If a file named .treeclipignore exists in the current directory, its non-comment lines will be used as the exclude list (one per line).
  - If neither --exclude nor .treeclipignore are provided, the default is: node_modules, .git

Example:
  treeclip --exclude=node_modules,.git,.cache

You can also create a .treeclipignore file like this:
  node_modules
  .git
  .next
  dist
  .vscode
